project.raw_data_visualization.gfs_build_visualization_map(date: str, hour: int, forecast: int, chart: str, extent: typing.List[int] = [13, 25, 56, 48], img_path: str = 'C:\\Users\\Kuba\\Desktop\\Projekt_GFS\\GFS-forecast-web-app\\project/../data/pics/0')¶

Prepares data, makes map with visualization and saves it to file.

Parameters:
  • date – given base date as string in format “YYYYMMDD”
  • hour – given base hour (UTC) as integer (available: 0, 6, 12, 18)
  • forecast – given forecast hour as integer (available integers 0-392)
  • chart – str - name of chart to be visualized (one from CHARTS or CHARTS_NONZERO)
  • extent – given extent as List[int] in format: [left_lon, right_lon, top_lat, bottom_lat]
  • img_path – path to directory where map will be saved.
